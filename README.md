# 雅 (Miyabi) - AI姓名判断ウェブアプリ

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fzukkyyoshida-arch%2Fmiyabi-ai)

伝統的な姓名判断（天格・人格・地格・外格・総格の五格）と、Gemini AIによる現代的な人生アドバイスを融合した、美しくレスポンシブな和モダンウェブアプリケーションです。


## 🌟 特徴
- **和モダン × Glassmorphism デザイン**: 漆黒と金色、薄紅色をベースにした、美しくアニメーションするUI。
- **Gemini AI 連携**: APIキーを設定することで、AIによる深く的確な姓名判断をその場で行えます。
- **安心のセキュリティ**: APIキーはサーバー等に送信されず、ブラウザの `localStorage` にのみ保存されます。
- **モック（デモ）モード搭載**: APIキーを設定しなくても、名前のハッシュ値からパーソナライズされた結果をシミュレート表示して体験できます。
- **タイピングエフェクト**: 診断結果が滑らかにタイピングされていく演出。

## 🛠️ 技術スタック
- フロントエンド: HTML5 / Vanilla CSS3 / JavaScript (ES6+)
- ビルドツール: Vite
- アイコン: Lucide Icons
- フォント: Google Fonts (Noto Serif JP / Outfit)

## 🚀 起動手順

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
npm install
```

### 2. 開発サーバーの起動
ローカルで動作確認を行うには、以下のコマンドを実行します。
```bash
npm run dev
```
起動後、ターミナルに表示されるローカルURL（例：`http://localhost:5173`）にブラウザでアクセスしてください。

### 3. 本番用ビルド
静的ファイルをビルドする場合は以下を実行します。
```bash
npm run build
```

## 📝 使い方
1. アプリを開き、名字と名前、それぞれのひらがなよみがなを入力します。
2. 必要に応じて生年月日や性別を入力します。
3. 「鑑定する」ボタンをクリックすると、鑑定が開始されます。
4. **AIによる本格的な鑑定を行いたい場合**:
   - 右上の設定アイコン（ギアマーク）をクリックします。
   - お手持ちの Gemini API キーを入力し、「モックモード」のチェックを外して保存します。
